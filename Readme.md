YouTube Clone 🎬

A simple YouTube-like web application that allows users to browse, search, and watch video content.
This project was built to understand the core concepts behind modern video platforms, including video listing, video playback, and responsive UI design.

🚀 Features

Browse a list of available videos

Watch videos using an embedded video player

Responsive user interface for desktop and mobile

Video thumbnails with titles and descriptions

Clean and intuitive user experience
